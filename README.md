# WhereMyMoneyLost (เงินหายไปไหน 💰)

**"WhereMyMoneyLost" (เงินหายไปไหน)"** คือแอปพลิเคชันบริหารจัดการการเงินส่วนบุคคลระดับพรีเมียมสำหรับ Android ที่พัฒนาขึ้นด้วยหลักการออกแบบสมัยใหม่ เน้นการสร้างความตระหนักรู้ทางการเงินผ่านภาพลักษณ์ที่สวยงามและใช้งานง่าย ตัวแอปจะช่วยตอบคำถามยอดฮิตที่ว่า *"เงินฉันหายไปไหนหมด?"* ด้วยแดชบอร์ดที่ชาญฉลาด ระบบติดตามอัตโนมัติ และ Widget หน้าจอโฮมที่ช่วยให้คุณเห็นสถานะการเงินได้ทันทีโดยไม่ต้องเปิดแอป

---

## 🚀 ฟีเจอร์หลัก (Key Features)

### 📊 แดชบอร์ดสรุปผลครบวงจร
- **ตระหนักรู้แบบเรียลไทม์**: ดูยอดใช้จ่ายวันนี้เทียบกับงบประมาณรายเดือนได้ทันที
- **กราฟิกแสดงความคืบหน้า**: ใช้แถบความคืบหน้า (Progress Bar) ตามมาตรฐาน Material 3 พร้อมระบบสีแจ้งสถานะ (เขียว/ส้ม/แดง)
- **เจาะลึกหมวดหมู่**: แสดงรายละเอียดการใช้จ่ายแยกตามหมวดหมู่ที่ปรับแต่งได้เอง (อาหาร, เดินทาง, ช้อปปิ้ง และอื่นๆ)

### 🎯 เป้าหมายการออม (Savings & Goals)
- **ออมเงินตามภารกิจ**: ตั้งเป้าหมายการออมเฉพาะด้านพร้อมระบุวันที่ต้องทำให้สำเร็จ
- **ติดตามความใกล้ชิด**: แสดงภาพกราฟิกว่าคุณเข้าใกล้ความฝันมากแค่ไหน
- **แจ้งเตือนความสำเร็จ**: ระบบจะส่งข้อความแสดงความยินดีทันทีเมื่อคุณออมเงินได้ครบตามเป้าหมาย

### 📅 จัดการบิลและรายจ่ายประจำ
- **ระบบติดตามบิลอัจฉริยะ**: จัดการบิลที่กำลังจะมาถึง และทำเครื่องหมายว่า "จ่ายแล้ว" ได้ง่ายๆ ในคลิกเดียว
- **แจ้งเตือนล่วงหน้า 1 วัน**: ไม่พลาดทุกยอดชำระด้วยระบบแจ้งเตือนอัตโนมัติ 24 ชั่วโมงก่อนถึงกำหนด
- **บันทึกรายจ่ายประจำอัตโนมัติ**: ระบบจะตรวจสอบรายการที่เป็นรายจ่ายรายเดือน (เช่น ค่าสมาชิก Netflix, ค่าเช่าบ้าน) และหักออกจากงบประมาณให้โดยอัตโนมัติทุกวัน

### 🔔 ระบบแจ้งเตือนอัจฉริยะ
- **เตือนเมื่อยอดใกล้เต็ม**: แจ้งเตือนเมื่อใช้จ่ายถึงระดับเปอร์เซ็นต์ที่กำหนด (เช่น 80% ของงบ)
- **เตือนเมื่อเกินงบ**: แจ้งเตือนทันทีเมื่อมีการใช้จ่ายเกินงบประมาณรวมหรือเกินงบในหมวดหมู่เฉพาะ
- **แดชบอร์ดค้างสถานะ (Ongoing)**: ตัวเลือกการแสดงผลยอดใช้จ่ายรายวันค้างไว้บนแถบแจ้งเตือนเพื่อให้ตรวจสอบได้รวดเร็วตลอดเวลา

### 🌓 ธีมและหน้าตาแอป (UI/UX)
- **โหมดกลางคืน (Dark Mode)**: รองรับการใช้งานโหมดมืดอย่างสมบูรณ์แบบเพื่อถนอมสายตา
- **สุนทรียภาพระดับพรีเมียม**: ดีไซน์สวยงามทันสมัยด้วย Material 3 (M3) พร้อมแอนิเมชั่นขนาดเล็ก (Micro-animations) ที่ทำให้แอปดูมีชีวิตชีวา

### 📱 Widget หน้าจอโฮมสุดล้ำ
- **รู้สถานะในพริบตา**: ดูยอดใช้จ่ายวันนี้และความคืบเนื้องบประมาณรายเดือนได้โดยตรงจากหน้าจอหลัก
- **สลับธีมอัตโนมัติ**: Widget จะปรับเปลี่ยนสีตามโหมดมืด (Dark Mode) ของแอปโดยอัตโนมัติ
- **เข้าถึงแอปได้รวดเร็ว**: แตะที่ Widget เพื่อเปิดแดชบอร์ดในแอปได้ทันที

## 🛠 เทคโนโลยีที่ใช้ (Tech Stack)

- **ภาษา (Language)**: [Kotlin](https://kotlinlang.org/)
- **โครงสร้าง UI**: [Jetpack Compose](https://developer.android.com/jetpack/compose) (การเขียน UI แบบประกาศค่าที่ทันสมัย)
- **สถาปัตยกรรม**: MVVM (Model-View-ViewModel) เพื่อการจัดการโค้ดที่สะอาดและเป็นระเบียบ
- **การจัดเก็บข้อมูล**: [DataStore](https://developer.android.com/topic/libraries/architecture/datastore) (ระบบจัดเก็บข้อมูลสมัยใหม่ที่รวดเร็วและปลอดภัย)
- **ดีไซน์**: Material 3 (M3) เพื่อความสวยงามระดับพรีเมียมและทันสมัยที่สุด
- **ระบบ Widget**: RemoteViews พร้อมระบบ Dynamic Resource ที่รองรับการสลับธีมแบบเรียลไทม์
- **การแจ้งเตือน**: Android Notification Manager พร้อมระบบ Custom Intent Filters

## 🛠 การติดตั้งและเริ่มต้นใช้งาน

1. **Clone repository**:
   ```bash
   git clone https://github.com/USER/WhereMyMoneyLost.git
   ```
2. **เปิดใน Android Studio**: แนะนำให้ใช้เวอร์ชัน Arctic Fox ขึ้นไป
3. **Build & Run**: รองรับอุปกรณ์ Android OS เวอร์ชัน 8.0 (API 26) ขึ้นไป เพื่อให้รองรับระบบ Notification Channels และ Home Screen Widgets
